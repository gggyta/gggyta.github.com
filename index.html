Github下载指定目录下文件

发表于 2016-06-08 | 分类于 git |
  1. 导航到github需要下载的文件夹目录下.
  2. 粘取出网页url
    例如: https://github.com/dongjunkun/DropDownMenu/tree/master/gradle/wrapper
  3. 将tree/master 改为trunk
  4. 到本地PC机任意文件夹中,点击右键用svn checkout…
    就可将指定文件copy到本地电脑
    https://github.com/dongjunkun/DropDownMenu/trunk/gradle/wrapper
